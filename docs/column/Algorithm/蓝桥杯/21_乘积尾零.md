 这题我第一眼是想着大数处理来着，我是个憨憨。。。

其实是把10拆成2和5，对每一个读入的数抽取2和5的个数，最后输出小的那一个就好了~

```c
#include <bits/stdc++.h>
using namespace std;
int main()
{
  // 请在此输入您的代码
  int cnt2=0,cnt5=0,x;
  for(int i=0;i<100;++i)
  {
    cin>>x;
    while(x%2==0) 
    {
      cnt2++;
      x/=2;
    }
    while(x%5==0)
    {
      cnt5++;
      x/=5;
    }
  }
  cout<<min(cnt2,cnt5);
  // cout<<31;
  return 0;
}
```
